## Installation

Add this line to your application's Gemfile:

```ruby
gem 'blinker'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install blinker

## Usage

Use `blinker start` command.

First argument is total minutes.
Second argument is last minutes.

    $ blinker start 5 1
